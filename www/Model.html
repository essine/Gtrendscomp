Model time series

 The 
'Model'
 module supports data modelling (see 
'Model' note
 at the page bottom).

 The 
'Data to model'
 menu allows to select data to model:

   
'Original data'
  - time series data obtained from google or imported from a CSV file.

   
'Remainder components (seasonal decomposition)'
  - is generated by seasonal decomposition of 'Original data'.
                      This menu item appears if seasonal decomposition was done in 'Detrend' program section.

   
'Differenced data'
  - differenced 'Original data'.
                      This menu item appears if differencing was done in 'Detrend' program section.

   
'Remainder components (smoothing)'
  - is generated by 'Original data' smoothing.
                      This menu item appears if smoothing was done in 'Detrend' program section.

                  
 Click 
'Model'
 button to model the selected data (
see 'Model' note
 at the page bottom). 
 The 
'Modeling data ...'
 message appears and blinks during the process. 

 The result will be presented graphically in the 
'Model plot'
 window, which will contain
 plots (
'Arima model residuals'
 and 
'Input data to model'
) and
 tables (
'Arima coefficients'
, 
'ARIMA(p,d,q)'
 and 
'Ljung-Box tests for model residuals'
).

 Under the plot window the plot options will appear:
 
'Select traces to plot:'
 option determines the traces to plot.
 
'Model coefficients to plot:'
 option indicates the trace for the 
'Arima coefficients'
 table. 

 The help window appears when 
'Help'
 button is pressed. Click the 
'Close'
 button to close the window.

 On an error, the error window could appear (see 
'Error window' warning
 at the bottom of the page). 

 
Notes:

  
'Model': Auto ARIMA modelling.
 
   The 'Model' program module uses the 'auto.arima()' function from the 'forecast' R package 
   which is detailed described by Rob Hyndman and coworkers (
www.otexts.org)
.

 
Warnings:
 
  
'Error window': Error window closing.

   To close an error window please click 
'Close please'
 button.
   Otherwise the window remains open even if the error disappeared.
