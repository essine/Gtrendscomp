Upload and import CSV formatted data

 The 
'Upload'
 module supports import of a CSV formatted trends data for further analysis.
 These could be custom data, data manually requested from Google Trends or data previously saved by the 'Data' module.
 The CSV file should contain columns with date and trends data, as well as a header row with column names.
 The column names will be used as labels for trends.

 The 
'CSV file format'
 can be set as :

   
'gtrendscomp CSV'
  - format in which 
'Date'
 program module saves data.

   
'custom CSV'
       - format which allows to skip the first few lines of a CSV file and/or 
                        replace column names (see 
'custom CSV' note
 at the page bottom). 
                        If 'custom CSV' item is selected the additional input fields appear: 

                        
'Skip n lines'
     - numeric input field to set the number of header lines to skip.

                        
'Set column names'
 - checkbox to activate column names edit option.  
                        If 'Set column names'checkbox is activated the text fields to enter column names 
                        and the checkboxes appear. To replace a column name the corresponding checkbox should be selected.

 To upload a CSV file use 
'Choose file'
 button (see 
'Choose file' warning
 at the bottom of the page). 
 The loaded data will be displayed as a table.

 The 
'Example CSV'
 checkbox is responsible for loading the example CSV file 
'gtrendscomp.csv'
. 
 The trends are equally scaled in the example (see 
'Data type'
 note at the bottom of the 
'Query'
 help page).
 Please keep in mind that the information about the query parameters is not available upon a CSV file upload. 
 If the example data are loaded, 
'Apply'
 button press can be skipped.

 Click 
'Apply'
 button to accept the data displayed. 
 The data will be presented graphically and accept for further analysis.

 The help window appears when 
'Help'
 button is pressed. Click the 
'Close'
 button to close the window.

 On an error, the error window could appear (see 
'Error window' warning
 at the bottom of the page). 

 
Notes:

  
'custom CSV': Google trends CSV file format.
 
   The 'custom CSV' file format can be used to upload a CSV file obtained from google trends service.
   For that, set 'Skip n lines' parameter to 1, select 'Set column names' checkbox and manually insert column names. 
   Please keep in mind that Google Trends CSV file format is subject to change by Google.
   If that happen, the recipe will not work without modification.
  
 
Warnings:
 
  
'Choose file': Reload problem.
 
   There is a problem to reload a CSV file without changing its name.
   Joe Cheng, a software developer on RStudio, suggest the following:
   "You might try clicking Browse and then Cancel - on my browser at least, 
   that causes the field to get blanked out, so you can choose a new file. It's not ideal, to say the least..."
   A less heavy solution to the above is to upload an another small CSV file first (e.g. example file 'gtrendscomp.csv').

  
'Error window': Error window closing.

   To close an error window please click 
'Close please'
 button.
   Otherwise the window remains open even if the error disappeared.
