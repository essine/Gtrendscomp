Download CSV formatted data

 The 
'Data'
 module supports data export to files in CSV format.

 Use the 
'Data in table'
 menu to select the data displayed in the table.
 The menu list includes all the data generated by application: input, detrended, model and comparison data.

 The 
'Load'
 button serves for loading the selected data as a CSV file (see 
'Data' note
 at the page bottom).
 The 
'Load All'
 button is reponsible for loading the whole data set as a zip file.

 The help window appears when 
'Help'
 button is pressed. Click the 
'Close'
 button to close the window.

 On an error, the error window could appear (see 
'Error window' warning
 at the bottom of the page). 

 
Notes:

  
'Data': Data format.
 
   The 'gtrendscomp' program data are saved in 'data' subdirectory in the RData file format:
   'td.RData' file contains input time series data (e.g. google trends data),
   'tdd.RData'file contains detrended data,
   'tm.RData' file contains model data,
   'tc.RData' file contains comparison results data,

   'Data' module saves and loads data in the CSV format. 
   The saved CSV data can be later uploaded via 'Upload' module and reanalyzed.

 
Warnings:
 
  
'Error window': Error window closing.

   To close an error window please click 
'Close please'
 button.
   Otherwise the window remains open even if the error disappeared.
